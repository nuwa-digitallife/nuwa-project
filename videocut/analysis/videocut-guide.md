# 视频剪辑经验

## 素材分析流程（铁律）

1. **密集抽帧** → 逐帧读包装文字/品牌/logo
2. **网搜确认** → 拿关键词搜商品，不靠猜
3. **建立事件因果图** → 哪个动作是因，哪个是果（开箱→取出→放置，不能倒）
4. **按因果逻辑排片** → 不是按素材编号排，也不是随意打乱，是按故事因果链排

## 排片规则

- **因果优先**：开箱在前，放置在后；发现在前，使用在后
- **同一事件的片段可以跨素材文件**：比如"开箱4前半"和"开箱4后半"之间可以插入其他素材
- **素材编号不等于剪辑顺序**：编号只是拍摄顺序，剪辑顺序由故事逻辑决定

## 标准交互模式（不需要用户提醒）

识别错误 → 改正 → 分析根因 → 记录经验。每次都这么做，不等用户问"为什么错了"。

## 选片规则（对照偏好模型）

- **每个片段问三个问题**：1. 画面有人在做事吗？ 2. 和视频主题相关吗？ 3. 是否与已选片段重复？
- **成果/展示镜头 ≤ 3秒**，必配字幕说明
- **重复动作用跳切压缩**：如码牛奶，只保留"开始→快满→满"三个节点，不要全程
- **无关物品不入镜**：窗帘纱不属于开箱主题，即使出现在素材里也不选

## 字幕review（剪辑后必做）

1. 每句字幕是否准确描述画面？
2. 是否与视频主题相关？（开箱视频里不出现与开箱无关的描述）
3. 全部字幕串联读一遍，是否构成完整故事？（有因有果，有开头有收尾）
4. 有没有重复/冗余？（如"不放过死角"和"不放过任何角落"不能同时出现）
5. 语气是否统一？（不能混入广告词式的"除醛90%以上"，要用口语化表达）
6. 开头有钩子吗？结尾有收束感吗？

## 教训记录

### V1-V3 教训（2026-03-01）
- V1：乱序 + 商品识别错（把活性炭认成窗帘布）→ 密集抽帧+网搜确认
- V2：因果断裂（先放炭包再开箱）→ 建因果图后再排片
- V3：没对照偏好模型选片 → 混入空镜（站着看楼梯10s）、无关镜头（窗帘纱）、重复动作（码牛奶36s）、广告词字幕
- **根因**：style_guide 只有泛泛规则没有逐帧拆解的偏好模型。必须先深度分析成品视频再剪辑
