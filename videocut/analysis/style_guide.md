# 视频风格指南 — LUNA COFFEE 系列

> 从成品视频（刷漆DIY 165.7s + 搞门头 77.1s）逐帧拆解提取

## 一、镜头偏好模型（核心）

### 镜头类型与占比

| 类型 | 占比 | 说明 | 时长 |
|------|------|------|------|
| **动手操作** | ~65% | 人在做事：拧螺丝/刷漆/搬运/拆箱/放置 | 3-15s/段 |
| **物品/成果展示** | ~15% | 展示做完的东西或选择的商品，**必配字幕说明** | 1-3s |
| **情感节拍** | ~5% | 里程碑时刻（灯亮了、招牌揭幕），**必配情感字幕** | 2-5s |
| **转场** | ~5% | 分屏转场/走到下一个位置 | 1-2s |
| **开头+结尾卡** | ~10% | 装饰标题卡/黑底手写结尾 | 2-5s |

### 铁律：什么镜头绝对不出现

- **无目的空镜**：不会拍一个空房间或空桌子，除非正在展示刚完成的成果
- **无意义走路**：不拍"走去某个地方"，到了就直接拍操作
- **与主题无关的物品**：视频主题是什么，镜头就围绕什么。无关物品不入镜
- **等待/发呆**：不拍站在那儿不动的画面
- **纯过渡空镜**：不用"先拍个环境再拍人"的建立镜头套路

### 成果展示镜头的规则

- **中间成果**：1-2秒，快速看一眼就切走
- **最终成果**：3-5秒，可以稍微多看一会儿
- **所有成果镜头必须配字幕**：说明这是什么、接下来要做什么、或表达情感
- 举例（刷漆DIY）：
  - 黑柜刷完 → "晾几天，等漆干了再贴螺钿"（1-2s，说下一步）
  - 花窗完成 → 直接切楼梯上漆（0转场）
  - 门面完工 → 3s全景，然后结尾卡

### 情感节拍的条件

只在**里程碑时刻**用：
- 灯第一次亮（"灯亮的那一刻，突然觉得，这个角落以后会有很多故事。"）
- 招牌揭幕（"软膜展开的瞬间还有点小激动😊"）
- 整体完工的夜景揭幕

**不是情感节拍**的：随便看了一眼窗户、展示一下窗帘——这些如果和主题无关就直接删。

## 二、剪辑节奏

### 场景时长分布

```
动手操作段：3-8s（简单动作）/ 8-15s（复杂过程如刷漆）
物品展示：1-3s
情感节拍：2-5s（稀有，一个视频1-2次）
无字幕呼吸：1-2s，场景间偶尔出现
```

### 转场手法

1. **分屏**：左边上一个任务成果，右边下一个任务工具/场景。两个视频都用了
2. **动作切**：人走到新位置 = 新场景开始，不需要空镜过渡
3. **时间跳切**：同一任务内可以跳过重复动作（如刷漆从1/4跳到3/4）
4. **完成标记**：用可爱贴纸（🐻"OK搞定"）标记完成，然后切新场景

### 长操作的处理

刷漆这类重复操作：
- 展示关键节点（开始→中间进度→完成），**不展示全程**
- 字幕辅助推进（"看着漆色渐渐填满，强迫症人格暗爽"）
- 衣服变化暗示时间流逝（灰外套→蓝工作服）

## 三、字幕系统

### 字幕视觉参数

- **字体**: 圆体/软体中文字体（非黑体，有圆润感）
- **颜色**: 白色 + 深色描边/阴影
- **位置**: 底部居中，约 85% 高度
- **大小**: 画面高度 3-4%
- **每行字数**: 5-15 字，短句口语化

### 字幕类型（6种）

| 类型 | 功能 | 举例 |
|------|------|------|
| **动作叙述** | 说正在做什么 | "开始门头上漆"、"抹布擦干净" |
| **设计说明** | 说为什么选这个 | "选择了昌迪加尔藤椅放窗口"、"选择了杜邦纸灯箱+暖光" |
| **自嘲幽默** | 翻车/辛苦/自嘲 | "上来就翻车"、"衣服就是废了"、"邻居路过估计都替我捏把汗" |
| **情感表达** | 里程碑时刻的感受 | "灯亮的那一刻，突然觉得，这个角落以后会有很多故事。" |
| **进度/完成** | 标记进展 | "看着漆色渐渐填满"、"OK搞定" |
| **互动钩子** | 开头结尾与观众互动 | "谁懂啊家人们"、"希望大家能喜欢"、"期待和大家见面~" |

### 字幕密度

- ~70% 画面有字幕
- 无字幕段不超过 2-3 秒（是呼吸，不是遗忘）
- **每个镜头至少有一句字幕赋予意义**——不能有"不知道这个镜头要表达什么"的情况

### 字幕review（剪辑后必做）

逐条检查：
1. 这句字幕是否准确描述了画面内容？
2. 是否与视频主题相关？（开箱视频里不应出现与开箱无关的物品描述）
3. 整体字幕串联起来是否构成完整故事？（有因有果，有开头有收尾）
4. 有没有重复/冗余的字幕？

## 四、装饰元素

- **开头装饰卡**（搞门头款）：暖色横幅 + 可爱贴纸(小熊等) + 钩子文案
- **完成贴纸**：任务完成时用贴纸标记（如 🐻"OK搞定"）
- **手绘标注**：箭头、圈圈指向细节（如"边边角角用刷子填满"）
- **emoji**：字幕中偶尔使用，不泛滥（"还有点小激动😊"）
- **结尾卡**：
  - 刷漆款：黑底 + 手写 "Bye Bye ❤"
  - 搞门头款：心形相框 + 手写 "我们下次再见。"

## 五、视频参数

### 成品参数
- **分辨率**: 960x544（刷漆）/ 568x320（搞门头）
- **帧率**: 30fps
- **编码**: HEVC (H.265)
- **音频**: AAC mono 44100Hz
- **时长**: 1-3 分钟

### 素材参数
- **分辨率**: 1280x720
- **帧率**: 30fps
- **编码**: HEVC
- **音频**: 现场环境音（无旁白）
