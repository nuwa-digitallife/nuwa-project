# 个人知识库

> 自动化的个人知识管理系统
>
> 创建日期: 2025-01-19

## 📚 概述

这是一个基于文章爬取、自动分类和交互式问答的个人知识库系统。

## 🗂️ 目录结构

```
knowledge_base/
├── 人工智能/          # AI、机器学习、深度学习
├── 商业/              # 商业策略、企业管理
├── 金融/              # 金融投资、经济分析
├── 个人成长/          # 个人发展、学习方法
├── 历史/              # 历史事件、文明演变
├── 哲学/              # 哲学思想、价值观
├── 文学/              # 文学作品、创作技巧
├── 未分类/            # 待分类或无法分类的文章
└── _index/            # 索引和配置文件
    ├── articles_index.json          # 全局文章索引
    └── classification_rules.json    # 分类规则
```

## 📝 文章结构

每篇文章都是一个独立的文件夹，包含：

```
文章标题/
├── article.md         # 文章正文（Markdown格式）
├── qa.md             # 问答记录
├── metadata.json     # 元数据（标题、作者、日期、标签等）
└── attachments/      # 附件（截图、PDF、图片等）
```

## 🏷️ 分类体系

### 主题分类（7个）
1. **人工智能** - AI、机器学习、深度学习相关技术
2. **商业** - 商业策略、企业管理、市场营销
3. **金融** - 金融投资、经济分析、财务管理
4. **个人成长** - 个人发展、学习方法、自我提升
5. **历史** - 历史事件、历史人物、文明演变
6. **哲学** - 哲学思想、思辨探讨、价值观
7. **文学** - 文学作品、文学评论、创作技巧

### 分类方式
- 自动分类：基于关键词匹配算法
- 手动调整：可随时修改文章分类
- 多标签支持：一篇文章可有多个标签

## 💬 问答功能

每篇文章都有独立的 `qa.md` 文件，记录：
- 阅读过程中的提问和回答
- 个人笔记和想法
- 完整的对话历史
- 时间戳记录

## 📊 统计信息

查看 `_index/articles_index.json` 了解：
- 总文章数
- 各分类文章数量
- 文章列表和元数据
