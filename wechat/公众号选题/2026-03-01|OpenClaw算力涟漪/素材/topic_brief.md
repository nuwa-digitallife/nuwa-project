# 选题：OpenClaw 和新模型发布引发的算力涟漪

## 核心主题
OpenClaw 以及各大模型厂商的新模型发布，带来的推理需求增长对算力和数据工厂的影响。在这些涟漪效应中，有哪些机会是人们没有关注到的？

## 调研方向

### OpenClaw 现状
- OpenClaw 最新进展（用户量、Agent 数量、API 调用量）
- OpenClaw 的商业模式和生态
- 使用 OpenClaw 的典型场景和用户画像

### 推理需求增长
- Agent/Agentic AI 带来的推理计算量增长（vs 传统 API 调用）
- 一个 agent 任务消耗的 token 量 vs 单次问答的倍数
- 行业预测：推理计算需求的增长曲线

### 算力供应链影响
- 推理需求增长 → GPU/芯片需求变化（推理 vs 训练的硬件差异）
- NVIDIA 的推理芯片路线（B200, GB200, Blackwell Ultra）
- 数据中心/数据工厂的建设热潮
- 能源需求：数据中心的电力消耗增长

### 被忽视的机会（重点）
- 推理优化软件（vLLM、TensorRT-LLM 等）的商业价值
- 边缘推理/端侧推理的机会
- 冷却技术（液冷、浸没式冷却）的需求爆发
- 电力基础设施（核能、SMR 小型核反应堆）
- AI 专用芯片创业公司（非 NVIDIA）
- 数据中心运维和管理工具
- 推理计费和成本优化工具

### Agent 创业公司的产业位置
- OpenClaw 等 agent 平台在算力供应链中的位置
- 他们是消耗算力的还是优化算力的？

## 人设
智子 — 涟漪观察者。从 OpenClaw 出发，沿着产业链向上追踪，找到二阶三阶效应和被忽视的机会。
