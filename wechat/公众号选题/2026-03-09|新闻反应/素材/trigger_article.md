今天有一个很出圈的事情：

腾讯推出一个免费帮忙代装龙虾的公益活动，现场非常的火爆。

然后我看到评论区有句话：**「一代人，有一代人的鸡蛋」**。

哈哈哈哈哈，绝。

![](https://mmbiz.qpic.cn/mmbiz_jpg/nR18mzFEw1F4RcvlLRbBvfMUl8FvicfKT0eAttTjBicyJibJWOQGnfZCDuciccBiaAs4vRamayc6GRMbtfOHu9u2B4PAaroPUhcT5ZSHhxGcDjzo/640?wx_fmt=jpeg&from=appmsg)

活动甚至 OpenClaw 创始人 Peter 也转发了。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/nR18mzFEw1EZbLj4Ah8Wn2TeibJqlY3rdnibbJm5BiaMhGBxzsg3KsI4VZZbqACnR9DqwGkCUJYBzsMOVXpXNIOVRiceVql57XCI07JFHptuibo4/640?wx_fmt=jpeg&from=appmsg)

不过还没装龙虾的朋友别着急，各种安装方式在这里[最全装龙虾教程](https://mp.weixin.qq.com/s?__biz=MzYyMTY1NDA0Nw==&mid=2247514677&idx=1&sn=2d5d09db9fd40b2ef8b54a43f5969985&scene=21#wechat_redirect)应该都讲清楚了。

然后，看到这篇文章的后台，也收到不少读者留言：

**OpenClaw 是装上了，模型也连好了，飞书也配完了。**

**然后就愣在那里，不知道让它干什么。**

不得不说，这些体验我太熟悉了。

两个月前我第一次接触 OpenClaw 的时候也是这样：我当时花了一下午部署成功，兴冲冲地打开对话框，让它帮我写个周报。结果也就那样，还不如问豆包。然后再让它帮我查个资料，它要么打开链接失败，要么给的不是我想要的，体验感极差，这还个人助理个啥啊。

后来，我想明白一个道理：**光有龙虾的「身体」还不够，你得给它装上技能。**

最近， GitHub 上有个项目悄咪咪爆火，专门在做这件事。

**项目名叫 awesome-openclaw-skills，是 GitHub 周增长的第十名。**

![](https://mmbiz.qpic.cn/sz_mmbiz_png/nR18mzFEw1ER5fbaTJX8KibcFjNEGMgjO0Hszib3QpmwwibSSGicSAVImwCgWPFPc6KDg8CW8aUNVpvmfrVcTpEP0JJeGQia3iaXhiaHwxLDOXjnJ8/640?wx_fmt=png&from=appmsg)

目前它已经收录了超过五千个龙虾装机必备 Skill，覆盖研究、开发、销售、日常管理等几乎所有你能想到的场景。

![](https://mmbiz.qpic.cn/sz_mmbiz_png/nR18mzFEw1HzrZGvP8MotlkZLmbYCvpzR5Na4qrY0Ec2X40EpFXeLEQlvLPxLjLp2sprOusPoMYMzj0ES3xnbcjiaEh2Iq6mic8vwtO5vUcYg/640?wx_fmt=png&from=appmsg)

选择适合你的 Skill，然后就可以让你的龙虾从不好用，变成 24h 离不开。

我花了一周时间把这个项目翻了个遍，发现了不少让人眼前一亮的玩法。

![](https://mmbiz.qpic.cn/sz_mmbiz_jpg/nR18mzFEw1GMIFgOiaUn8pQnnLlWLERpbVZYg94usMPIX7cfpVKdz1CruCMib5Zt77Udqr0y840b7oBCUPZxHRp6TOY0jekcXicxHSSnvnOwZg/640?wx_fmt=jpeg&from=appmsg)

这些 Skill 的价值在于，它们把「搜索」这件事拆解成了一套完整的工作流。

而龙虾很擅长的就是做研究。齐了，从研究学者直接升级为研究老炮：

![](https://mmbiz.qpic.cn/mmbiz_png/nR18mzFEw1Gk8R8BJHwoGCD6ZR6Aicboef6agrpEsgwMNZarZq4CGCxAJACd4u27ntJk4TCrp9JF0PF8kZMWiclN4l4hm4bT4AJ1Y1iah1FvwU/640?wx_fmt=png&from=appmsg)

![](https://mmbiz.qpic.cn/sz_mmbiz_png/nR18mzFEw1G5gWPt0QjKwvGKI2VLyjliaZ4fbW1ONN2iaW2akuBv3xXJ2YZWjKTfvtfHPicicpK9YLC106EmWicHwibRkYxeAEQXseSz1vhAPX7sA/640?wx_fmt=png&from=appmsg)

花了一周时间泡在这个项目里，我发现了一些值得关注的趋势：

**1、AI Agent 正在长出手和脚，直接连接真实世界。**

有非常多的 Skill 提供了让你的龙虾可以浏览网页、收发邮件、管理日历、阅读文档等能力。比如像控制 Mac、代为购物（AgentPay）这样的技能，更是让代理能直接操作物理世界和完成交易。

如果把你的 Mac Mini/云厂商的服务器看成是龙虾的身体。

那么龙虾给这个身体赋予了一个聪明的大脑（取决于你用的模型）和灵魂。

然后 Skill 给龙虾装上了手和脚，让龙虾从虚拟世界踏入现实。

**2、围绕个人智能体的生态，已经在逐渐形成。**

从项目的分类就能看出来。Skill 的覆盖范围已经从传统生产力工具，延伸到社交网络、游戏、健康管理，甚至约会平台。金融、医疗、法律这些专业领域都有相应技能在涌现。

针对不同行业、不同场景，几乎每个垂直领域都有数百个技能在冒出来。

甚至还出现了专门给龙虾交流的各种各样的社区和 Agent 沟通协议：

![](https://mmbiz.qpic.cn/sz_mmbiz_png/nR18mzFEw1GrQavGK9YByVJUFYdxVlTPibedGcQiaLwLPicNqJovU1mZyic3yQniaZF1OfLlEkY7drWoOwtiaYwXOOyj8avafo8O81f3AtCnXLPPA/640?wx_fmt=png&from=appmsg)

这意味着什么？

也许用不了多久，你的龙虾就是你在虚拟世界的分身。

它掌握的 Skill 决定了它的能力边界：

**技能越多、越垂直、效果越精细，它就能在你不在的时候，帮你悄咪咪做更多事情。**

而这会颠覆我们的日常生活：

赚钱、交友、写代码、管日程，这些以前必须亲力亲为的事，正在变成可以委托给龙虾的任务。
