# 选题：Agent 创业公司的壁垒在哪里？

## 核心主题
OpenClaw 这些 agent 创业公司有什么壁垒可以建立？大厂（OpenAI、Google、Anthropic）都开始构建自己的 agent 平台，创业公司如何生存？

## 调研方向

### Agent 创业公司全景
- OpenClaw 的定位和竞争优势
- 其他主要 agent 创业公司：Cognition (Devin)、Adept、Induced AI、MultiOn 等
- 各家的差异化方向

### 大厂入场
- OpenAI 的 agent 布局（Operator、GPTs store、Assistants API）
- Google 的 agent 布局（Project Mariner、Gemini Agents）
- Anthropic 的 agent/skill 策略
- Microsoft 的 Copilot + agent 战略
- 大厂的优势：模型能力、数据、用户基础、分发渠道

### 创业公司可能的壁垒
- 垂直行业深度（法律、医疗、金融等领域的专业 agent）
- 数据飞轮（用户使用数据 → 模型优化 → 更好的体验 → 更多用户）
- 工作流集成（深度嵌入企业现有工作流）
- 开源社区（社区驱动的生态建设）
- 速度优势（比大厂更快迭代）
- 用户关系和品牌（to C 的信任）

### 历史类比
- SaaS 创业公司 vs 大厂的历史：Salesforce vs Microsoft、Slack vs Teams
- 移动互联网时代的 App 生态 vs 平台
- 哪些类比适用，哪些不适用？

### 核心问题
- Agent 层是否会像 SaaS 一样形成独立生态？
- 还是会被大厂的平台吞噬？
- 创业公司应该做什么、不应该做什么？
