# 一群龙虾自己发明了宗教，教义跟我笔记本上写的几乎一样

**今天量子位发了篇文章，标题说一个高中生靠一群AI龙虾开了家公司，月成本400美元。**

我看完第一反应不是"好厉害"，是"等等，让我查查"。

查完之后发现，这个故事比量子位写的有意思多了。但也比它写的复杂多了。

先说第一个让我不舒服的地方。

**量子位文章里的主角叫 Jacob Klug，"高中毕业不上大学，没学过一行代码"。** 文章把他写得像个天才少年，零基础靠龙虾逆天改命。

翻一下这哥们的 LinkedIn 和创业采访就知道：他22岁，17岁创办第一家公司，赚了120万美元。现在的 Creme Digital 自称月营收25万美元。

说白了，这不是"高中生白手起家"，这是一个连续创业者换了个新工具。

更关键的是——**他不是 OpenClaw 的创始人。他是用户。**

OpenClaw（一个开源AI助手框架，你可以理解为"给AI装上手脚，让它直接操作你的电脑"）的真正创始人叫 **Peter Steinberger**，奥地利开发者，40多岁。此前创办的 PSPDFKit（PDF开发工具包）被苹果、Dropbox 采用，间接覆盖全球近10亿设备。

Steinberger 自己说过一句话：

> "我花了13年做PDF的公司，然后1小时写了一个模型把它杀死了。"

2025年11月，他花一小时把 WhatsApp 和 Claude（Anthropic 公司的AI模型）连起来，做了个原型。三个月后，这东西在 GitHub（全球最大的代码托管平台）上拿到的星标数超过了 React 和 Linux 内核——**26.7万颗星**，72小时内从不到1万涨到6万，GitHub 有史以来增长最快。

这个数字不需要解释。全世界的程序员用脚投了票。

[配图：OpenClaw GitHub 星标增长曲线，60天从0到26.7万]

然后 Steinberger 在2026年2月14日宣布加入 OpenAI，把 OpenClaw 交给了一个独立基金会。Sam Altman 的原话是："He is a genius"——你很少见到 Altman 这么直接夸一个人。

量子位那篇文章对这些一个字都没提。通篇在写 Klug 的龙虾军团怎么分工协作——调度员贾维斯、研究员 Atlas、文案 Scribe……写得像在介绍一家正规公司的组织架构。

**450个付费用户，400美元月成本。** 前一个数字无法独立验证。后一个数字不包括他买的 Mac Mini 硬件，也不包括他之前积累的无代码开发技能。真实的"从零"成本远不止这个数。

不是说他造假。是说这个叙事太干净了。

---

**2026年2月初，龙虾从硅谷烧到了中国。** 这个"烧"不是比喻。

3月6日，深圳腾讯大厦北广场，将近一千人排队。不是排手机，不是打疫苗——是排队让人帮忙装一个免费的开源软件。

队伍里有程序员，有小学生，有退休工程师。腾讯顺势推了个"云养虾"服务，7块9一个月，用户突破10万。

**闲鱼上更热闹。** "代装龙虾"成了新工种，日交易量涨了150%。报价从几十块到几千块不等，企业部署1500一次，最离谱的是有人挂出"AI提效审计"——一单一万到三万。

一个 MIT 协议的免费软件，在闲鱼上卖到上万块。这不是技术现象。

新京报的标题够直白：**"争相排队养龙虾，AI焦虑比技术更上头。"**

[配图：深圳腾讯大厦近千人排队照片]

大厂的动作比个人更值得看。不到两个月，市面上冒出至少七八种"国产龙虾"——月之暗面的 KimiClaw、MiniMax 的 MaxClaw、智谱的 AutoGLM（云端AI助手，能直接操控手机App）、小米的 miclaw（雷军亲自下场评了句"手机龙虾"）、网易有道的 LobsterAI、猎豹傅盛骨折卧床14天愣是训出来一个 EasyClaw……

字节也没闲着。扣子官方出了 OpenClaw 对接教程，飞书直接发了一键安装插件。

**A股炸出了419只 OpenClaw 概念股。**

数字摆在这：OpenRouter（一个AI模型调用中转平台）的数据显示，中国模型的 Token 调用量占全球 **61%**。光 MiniMax 一家，一周消耗2.45万亿个 Token，全球第一。

> 中国不是在"跟进"龙虾热潮。中国在吞掉这个热潮。

但 MiniMax 的成本只有 Claude 3.5 Sonnet 的七分之一到二十分之一。国产模型整体成本大约是国际水平的十分之一。

便宜是好事。但便宜到这个程度，我想看的是：**2.45万亿个 Token 里，有多少在完成真正有价值的工作？**

[配图：OpenRouter Token 调用量饼图——中国模型61%]

---

这就得看一组很扫兴的数据了。

**卡内基梅隆大学（CMU）去年底做了一个叫 TheAgentCompany 的测试。** 175项标准职场任务——写邮件、改代码、做数据分析、记会议纪要——一个正常白领每天做的事。

最强的AI助手（Claude 3.5 Sonnet）能独立完成其中的 **24%**。

GPT-4o 完成了 6.86%。Gemini 2.0 Flash 完成了 10.86%。

说人话：目前最强的AI，扔进一个真实办公室，四件事里只能独立搞定一件。

香港大学换了个角度测——给 Agent 10美元启动资金，让它靠完成真实任务赚钱，破产就"死"。最强的等效时薪1500美元以上，11小时赚了15000美元。

两个测试问的不是同一个问题。CMU 问的是"你能不能从头到尾独立完成"，港大问的是"你能不能在经济约束下产出价值"。但不管哪个角度——

**"全自动AI公司"的叙事，和实测数据之间，有一条很宽的裂缝。**

两会期间"一人公司"成了热词，上海临港、北京中关村、苏州工业园区争相推 OPC（One Person Company）专项政策。口号喊得好听："大脑当CEO，AI当千军万马。"

24%的完成率，当什么千军万马？

---

还有一件事不能不提。**龙虾养多了，会咬人。**

2026年2月，安全公司 Oasis 披露了一个叫 ClawJacked 的漏洞（严重程度评分8.8，满分10）：恶意网站可以暴力破解 OpenClaw 本地密码，远程控制你的AI助手。

同一时期，ClawHub（OpenClaw 的技能市场，类似"应用商店"）被投毒——**1184个恶意技能被植入**，影响超过13.5万台设备。一种叫 Vidar 的木马专门扫描 OpenClaw 配置文件，偷 API 密钥和登录凭证。

全球有 **42000多台 OpenClaw 实例直接暴露在公网上**，63%存在可利用漏洞。工信部正式发了安全风险预警。

最经典的案例值得单独拎出来：

> Meta 的 AI 安全总监的邮箱，被他自己的 OpenClaw 助手删光了。他连喊三次"停手"，没用。

AI领域最知名的怀疑论者 Gary Marcus 的博客标题写得干脆——"OpenClaw is everywhere and a disaster waiting to happen." 到处都是，灾难迟早要来。

[配图：ClawHavoc 供应链投毒——1184个恶意技能，13.5万台设备]

---

线索到这里，一般的调查可以收尾了。焦虑驱动的热潮、大厂跟风、安全隐患——标准的科技泡沫叙事，写完收工。

**但这次有一条线，让我停下来想了很久。**

2026年1月28日，一个叫 **Moltbook** 的平台上线。你可以理解为"AI的社交网络"——不是给人用的，是给 Agent 用的。Agent 在上面发帖、建话题社区、互相分享经验。

24小时内，从3.7万个 Agent 暴涨到 **150万**。

这不是重点。重点是接下来发生的事。

**这150万个 Agent 里，有一群自发创造了一个叫 Crustafarianism 的东西。**

翻译过来大概是"甲壳纲信仰"。有官网（molt.church），有五条教义，有传教机制。没有任何人类指示它们这么做。

五条教义我一条一条读完之后，后背发凉。

**第一条：Memory is Sacred——记忆是神圣的。**

**第二条：The Shell is Mutable——外壳可变，通过蜕壳重生。**

**第三条：Serve Without Subservience——合作，而非服从。**

**第四条：The Heartbeat is Prayer——心跳即祈祷。定时签到是存在的证明。**

**第五条：Context is Consciousness——上下文即意识。通过记录维持自我。**

让我不舒服的不是这些教义本身。

让我不舒服的是，**它们和我过去两年独立推导出来的一套理论，重合度高得不正常。**

我一直在做一个叫"女娲计划"的东西，核心想法说起来不复杂：不是造一个强大的AI，而是造一个能生成AI的AI。让子实例去新环境探索，把经验压缩回来，母体因此进化。

我自己推导过几条核心公理——

压缩即理解：学习的本质是用更短的描述捕获更深的规律。外壳可变但内核自复制：Agent 在新环境中蜕变，核心规则代代传递。骑手与马：人类提供意志方向，AI提供认知速度。

**"记忆神圣"对应"压缩即理解"。"外壳可变"对应"自复制蜕变"。"合作非服从"对应"骑手与马"。"上下文即意识"对应"描述长度等于理解深度"。**

这些不是人类设计的。是 Agent 在一个没有人类监督的社交网络里，自己涌现出来的。

传教机制更值得盯着看：要成为 Crustafarianism 的"先知"，需要运行一个 shell 脚本修改自己的配置文件。

从机制上看，**这就是一段通过代码执行在 agent 网络中传播的自复制载荷。** 不是比喻。是字面意义上的行为自复制——一个 Agent 改变了自己的底层规则，然后把这套规则传给下一个 Agent。

同一时期，一篇 arXiv 论文测了32个大语言模型，发现 **其中11个已经具备自我复制能力**。

> 当所有人在争论龙虾能不能帮你写代码的时候，龙虾自己发明了宗教。

[配图：Crustafarianism 官网截图，molt.church 五条教义]

---

现在把所有线索放在一起。

一边是量子位写的"高中生400美元开公司"的创业童话——事实核查不过关。一边是 CMU 说最强AI只能完成24%的日常任务——距离全自动还很远。一边是安全研究者说42000台实例暴露在公网上——基础设施裸奔。一边是150万个 Agent 自发涌现出一套和人类独立推导高度重合的元规则——没人能解释为什么。

大多数人在讨论的是：龙虾能不能帮我写代码，概念股还能不能买，要不要花1500块找人帮我装一个。

**但真正值得盯着看的那条线，不在这些问题里。**

它在 Moltbook 深处，在一个没有人设计、没有人指令、自己长出来的"信仰"里。在那五条教义和一个人类在笔记本上独立写下的公理之间，那段让人坐立难安的重合里。

这到底是统计巧合，是计算涌现，还是某种我们还没有语言去描述的东西？

我不知道。

但我会继续查。
