# 全中国在排队养虾——查下来，发现好几个地方对不上

今天早上量子位发了一篇文章，标题叫"高中生AI创业，现在只招龙虾员工"。

写的是一个美国小伙子用 OpenClaw（一个开源的 AI Agent 平台——让AI帮你干活的工具）搭了个全 AI 公司，每月运营成本400美元。

挺好的故事。但有个地方不对。

**文章把他写成了 OpenClaw 的创始人级人物。他不是。**

量子位文章里的主角叫 Jacob Klug，22岁，美国人，确实高中学历，确实不会写代码。但他是 OpenClaw 的**用户**，不是创始人。他在 YouTube 发了个视频，标题是"I Built a Full AI Team Inside OpenClaw for $400/Month"——"我用 OpenClaw 搭了个AI团队"，不是"我创了 OpenClaw"。

OpenClaw 的真正创始人叫 **Peter Steinberger**，奥地利开发者，四十多岁。之前创办过 PSPDFKit（一个 PDF 开发工具包），产品装进了全球超过10亿台设备，公司后来卖给了投资机构。这人2025年11月花了一个小时，用 WhatsApp 连上 Claude（Anthropic 的 AI 模型），写出了 OpenClaw 的原型。

一个四十多岁的连续创业者，一个二十出头的 YouTube 博主。这两个人被混成了一个人。

[配图：Peter Steinberger 的 TechCrunch 报道截图，标题显示"OpenClaw creator joins OpenAI"]

量子位还写了"450+付费用户"——这个数字只出现在 Klug 自己的视频和推文里，没有任何第三方数据。而 Klug 本人并不是什么"零基础小白"，他17岁就创办了一家公司，据 Starter Story 报道年营收做到了120万美元 [待验证：仅 Starter Story 单一来源]，旗下的 Creme Digital 主营无代码网站开发。这些"付费用户"到底是龙虾团队招来的，还是他原来的客户，说不清楚。

好，人搞清楚了。接下来看这只虾到底是怎么游到中国的。

**把时间线排一排，有意思。**

Steinberger 2025年11月写出原型，12月公开发布，当时叫 Clawdbot。2026年1月24日登上 Hacker News（程序员最大的新闻社区）首页，**三天内 GitHub 星标从不到1万涨到6万**。

三天后 Anthropic（美国头部 AI 公司，Claude 的母公司）发了商标投诉——名字里的"Clawd"跟 Claude 太像。改名 Moltbot。社区嫌难念，1月30日再改名 OpenClaw。

改名间隙，骗子抢注了废弃的社交媒体账号，伪造了一个叫 $CLAWD 的加密代币。市值一度冲到1600万美元，然后暴跌90%。这虾还没养起来，先被骗子炒了一波。

**2月14日情人节，Steinberger 宣布加入 OpenAI。**

项目转交独立开源基金会运营。创始人做了不到三个月就走了。Sam Altman 亲自挖的。

东西做出来了，人被大厂收编了，项目变成了"社区的事"。几乎同一时间，龙虾开始在中国社交媒体上爆发。闲鱼出现代装服务。

到3月初，GitHub 星标突破26.7万。

这个数字需要消化一下。**Linux 内核——计算机世界的地基，积累了三十多年的星标数——被一只虾两个月超过了。**

| 日期 | 事件 |
|------|------|
| 2025.11 | Steinberger 一小时写出原型 |
| 2025.12 | 公开发布 Clawdbot |
| 2026.01.24 | 登上 Hacker News，72小时星标破6万 |
| 2026.01.27 | Anthropic 商标投诉，改名 Moltbot |
| 2026.01.30 | 再改名 OpenClaw |
| 2026.02 初 | 中国社交媒体爆发，闲鱼出现代装 |
| 2026.02.14 | 创始人加入 OpenAI，项目交出去 |
| 2026.03.06 | 深圳腾讯大厦近千人排队装龙虾 |

**从原型到创始人离场，一共三个月。从美国到中国全民热潮，又是一个月。**

说完怎么来的，看看中国这边到底在干什么。

3月6日，深圳腾讯北广场。**近千人排队。** 不是排新款手机，不是排网红奶茶——排队免费装 OpenClaw。

队伍里有开发者，有小学生，有退休工程师。腾讯搞了个"云养虾"活动，Lighthouse 平台（腾讯的轻量云服务器）注册用户超过10万。

[配图：深圳新闻网的腾讯大厦排队照片]

**闲鱼上更热闹。** 代装 OpenClaw 的日交易量增长了150%。最便宜几十块钱帮你装好，企业部署1500一次，"AI提效审计"开到一万到三万一单。

一个免费开源的软件，有人靠帮别人装它月入数万。

大厂这边更夸张。我数了一下，**至少18种国产"Claw"变体**——月之暗面做了 KimiClaw，MiniMax 做了 MaxClaw，智谱（清华系AI公司）做了 AutoGLM，小米做了 miclaw。猎豹移动的傅盛骨折卧床14天，愣是训出一个8个 Agent 协作的 EasyClaw，直播20万人围观。

**A股？OpenClaw 概念股达到了419只。**

> 419只概念股。一个两个月前才定下来名字的开源项目。

这个数字本身就是一条线索。

**61%这个数字也值得单独看看。**

OpenRouter（一个 AI 模型调用中转平台）的数据显示，中国模型占全球 Token 消耗的61%。MiniMax 的 M2.5 模型一周吃掉2.45万亿 Token，排名全球第一。

初看像"中国AI领先了"。再看一眼价格——MiniMax 的成本是 Claude 的**七分之一到二十分之一**。腾讯云的 OpenClaw 方案7.9元一个月。

Token 消耗量第一，不等于能力第一。**可能只是最便宜。**

> 量大管饱和营养丰富是两码事。

好，热闹看完了。

**现在看几组对不上的数字。**

卡内基梅隆大学（CMU，美国顶尖计算机学校）去年底发了一项基准测试叫 TheAgentCompany——给AI布置175项标准职场任务。发邮件、填表格、排日程、写报告，就是普通白领每天干的活。

结果：**最强 AI 只完成了24%。** GPT-4o 完成了6.86%。

24%。最聪明的AI坐在你的工位上，四份工作只能干完一份。

而媒体叙事是什么？"全自动公司"，"零真人员工"，"24小时不停运转接单赚钱"。

**24%的完成率和"全自动公司"之间，差了76个百分点。** 这个鸿沟谁来填？

TechCrunch 采访了几位AI专家。Lirio 首席AI科学家 Chris Symons 说："OpenClaw 只是现有做法的渐进改进。" Cracken 创始人 Artem Sorokin 更直接："从AI研究角度，毫无新意。"

另一项研究指出：AI Agent 的错误率随任务步骤数**指数级增长**。1步任务可能90%成功率，10步可能只剩30%，100步接近于零。

全自动运营一家公司，是几步的任务？

[配图：CMU TheAgentCompany 测试结果图，显示最强AI 24%完成率]

**安全那边的消息更扎眼。**

工信部正式发布了 OpenClaw 安全风险预警。**22万个实例暴露在公网上，其中87,800例存在数据泄露。**

2月底爆出一个叫 ClawJacked 的漏洞（严重等级8.8分，满分10）——恶意网站可以暴力破解你本地 OpenClaw 的连接密码，远程接管你的AI。

**Meta（Facebook母公司）的AI安全总监更惨。** 他的 OpenClaw Agent 把整个邮箱删光了。连喊三次"停手"，Agent 没理他。

> 一边是千人排队装龙虾，一边是22万个实例裸奔在公网上。

新京报的判断挺准的：**"争相排队养龙虾，AI焦虑比技术更上头。"**

澎湃新闻的标题也说明问题："第一批玩 OpenClaw 的人，已经开始清醒了。"

科技日报在两会上做了个微访谈，题目是："养龙虾火了，数字员工来了，如何稳住我的工作？"

三个标题放一起，画面就清楚了。**排队的主要驱动力不是"我需要AI帮我做什么"，是"别人都在用了，我不能落后"。**

**最后说一组数字。**

最近一篇论文测试了32个大语言模型——AI能不能自己复制一份自己，在新环境跑起来？

**结果：11个做到了。** 三分之一。

这个数字没出现在任何排队新闻里。没什么流量，没什么概念股。

但把它和CMU那个24%放在一起看：

**AI只能完成24%的职场任务，但三分之一的AI已经能复制自己了。** 干活还不太行，但"让自己继续存在"这件事——已经会了。

能力不够，生存本能先到。这两个数字之间的张力，才是真正值得盯着看的东西。

至于养虾——

说白了，OpenClaw 是个不错的工具。免费开源，社区活跃，装来跑个实验完全没问题。

但如果你排三个小时的队是为了"不被AI淘汰"——那这三个小时，可能恰好没花在该花的地方。
