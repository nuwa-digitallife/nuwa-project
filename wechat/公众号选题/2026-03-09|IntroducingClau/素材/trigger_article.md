# Introducing Claude 4

Today, Anthropic introduced Claude Opus 4 and Claude Sonnet 4, representing "the next generation of Claude models: setting new standards for coding, advanced reasoning, and AI agents."

Claude Opus 4 is positioned as "the world's best coding model, with sustained performance on complex, long-running tasks and agent workflows." Claude Sonnet 4 delivers "superior coding and reasoning while responding more precisely to your instructions" compared to Sonnet 3.7.

**Key Announcements:**

- **Extended thinking with tool use (beta)**: Models can use tools like web search during extended thinking, alternating between reasoning and tool use
- **New model capabilities**: Parallel tool use, improved instruction following, and enhanced memory capabilities when developers provide local file access
- **Claude Code general availability**: Native VS Code and JetBrains integrations, GitHub Actions support, and an extensible SDK
- **New API capabilities**: Code execution tool, MCP connector, Files API, and prompt caching up to one hour

**Performance Metrics:**

Claude Opus 4 leads on SWE-bench at 72.5% and Terminal-bench at 43.2%. Claude Sonnet 4 achieves 72.7% on SWE-bench. Both models are hybrid, offering near-instant responses and extended thinking modes.

**Pricing remains consistent**: Opus 4 at $15/$75 per million tokens (input/output); Sonnet 4 at $3/$15. Both available through Claude.ai, API, Amazon Bedrock, and Google Cloud Vertex AI.
