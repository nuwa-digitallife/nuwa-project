# 选题：中国 AI Token 调用量超过美国

## 核心主题
中国 AI 的 token 调用量（API 调用量）已经超过美国公司。探讨低成本优势对中美 AI 竞争格局的影响，以及这个数据点背后的产业链涟漪效应。

## 调研方向

### 核心数据
- 中国 AI token 调用量超美的具体数据来源（哪家报告？什么时间点？）
- MiniMax、DeepSeek、通义千问等中国模型的 API 调用量数据
- OpenAI、Anthropic、Google 的 API 调用量对比
- token 价格对比：中美模型的定价差异

### 低成本优势的来源
- 中国模型的成本结构（蒸馏、开源、推理优化）
- DeepSeek 的低成本路线是怎么实现的
- 硬件差异：H800/H20 vs H100/B200 对推理成本的影响
- 开源模型（Qwen、DeepSeek）降低了准入门槛

### 产业链涟漪（重点）
- 一阶效应：更多应用开发者使用中国模型 → 中国 AI 应用生态壮大
- 二阶效应：推理需求增长 → 中国算力基础设施投资增加 → 哪些公司受益？
- 三阶效应：低成本 AI → 渗透到更多传统行业 → 中国在 AI 应用层的先发优势？
- 反面：调用量大 ≠ 价值高，需要区分"量"和"质"

### 地缘影响
- 美国芯片制裁本意是限制中国 AI → 结果可能反而促进了低成本创新
- 中国 AI 在应用层 vs 基础研究层的不同定位
- 这对全球 AI 产业意味着什么？

## 人设
智子 — 涟漪观察者。一个数据点（token调用量超美）→ 追踪产业链传导 → 找到被忽视的信号和机会。
