# 机会库添加规则

> 每次新增投稿机会到数据库前，必须通过以下检查清单。

## 核心约束

### 🔴 征文/比赛类 — 首发限制（必检）

**定义**：有截止日期、有奖项、由协会/出版社主办的征文活动

**隐藏的首发限制**：
- 95% 的征文比赛都要求"原创首发"（虽然不是所有都明确写出来）
- 一旦在某个征文投稿并获得反馈，就无法再投其他征文（违反首发承诺）
- **一篇文章 = 只能投一个征文，加多个期刊**

**🔴 征稿要求原文必须单独保存**：
- 征文的原创首发条款往往隐藏在简章深处，措辞复杂，摘要不够用于发送前核查
- **原文存储位置**：`投稿机会/征稿原文/{征文名}.md`（完整征稿简章/版权条款原文）
- **机会库中只存关键摘要 + 关联路径**，格式：`见 投稿机会/征稿原文/xxx.md`
- 发送前，人工打开原文文件逐条核查，不能只看机会库摘要

**检查点**：
- [ ] 征文简章中是否明确写"原创"、"未发表"或"首发"？
- [ ] 投稿须知是否有版权声明或独占承诺？
- [ ] 咨询邮件确认：如果同时投多个征文，是否违规？
- [ ] **关键问题**：投这个征文后，还能投其他征文吗？

**结论标记**：
- 🟢 **绿色** — 明确允许一稿多投（极少）
- 🟡 **黄色** — 未提及，需邮件确认
- 🔴 **红色** — 明确要求原创首发（无法同时投其他征文）

### 🔵 期刊/报纸 — 可通常一稿多投

**定义**：无截止日期、常年征稿、按期出刊的文学期刊/报纸副刊

**特点**：
- 通常允许一稿多投（发表后需标注转载）
- 同一篇文章可投《红豆》+《广西文学》+《广西民族报》（但一发后需告知其他编辑）

**检查点**：
- [ ] 是否有明确的截止日期？（无 = 期刊）
- [ ] 投稿须知中是否提及"一稿多投"或"版权转让"？
- [ ] 标准投稿邮箱是否在官网查证？

## 添加新机会的完整流程

### Step 1：信息采集
| 字段 | 获取方式 | 优先级 |
|------|---------|--------|
| 官方网址 | 搜索 + 官网验证（不信任转发） | ⭐⭐⭐ |
| 截止日期 | 简章第一句话 | ⭐⭐⭐ |
| 体裁和字数 | 参赛要求 | ⭐⭐⭐ |
| 投稿邮箱/方式 | 投稿须知 | ⭐⭐⭐ |
| 奖金/稿费 | 奖励说明 | ⭐⭐ |
| **首发/版权声明** | 投稿须知的第 2-3 段 | ⭐⭐⭐ |
| **征稿要求原文** | 完整复制征稿简章到 `投稿机会/征稿原文/{征文名}.md` | ⭐⭐⭐（征文类必须） |
| **类型分类** | 根据截止日期判断：期刊 vs 征文 | ⭐⭐⭐ |

### Step 2：一稿多投兼容性判断

```
┌─ 这个机会是什么类型？
│
├─ 🟢 期刊/报纸（无截止日期）
│  └─ ✅ 可与 N 个其他期刊同投
│  └─ 🟡 但投前需邮件问"一稿多投政策"
│
├─ 🔴 征文/比赛（有截止日期）
│  └─ ❌ 只能选一个征文投
│  └─ ✅ 可加投 M 个期刊（但发表后需告知）
│  └─ 📋 在数据库中标记为"红色不兼容"
│
└─ 🟡 不确定
   └─ 邮件确认首发政策
   └─ 确认前不添加或标记为"待确认"
```

### Step 3：入库前的金线检查

添加到机会库前，核对：

- [ ] **主办单位** — 是否是真实合法的机构？（防止诈骗征文）
- [ ] **联系方式** — 邮箱是否与官网一致？
- [ ] **字数范围** — 是否能容纳现有素材？
- [ ] **最重要：首发限制** — 明确分类为"期刊"或"征文-红/黄/绿"
- [ ] **征文类专项**：已将征稿要求原文保存到 `投稿机会/征稿原文/{征文名}.md`
- [ ] **征文类专项**：机会库"征稿要求原文"字段已填写关联路径

### Step 4：数据库字段

每个机会必须包含：

```markdown
| 项 | 值 |
|---|---|
| **主办** | ... |
| **类型** | 期刊 / 征文(红-独占) / 征文(黄-待确认) |
| **⚠️ 首发限制** | 明确描述：原创首发/允许一稿多投/未提及 |
| **征稿要求原文** | 期刊类：N/A；征文类：见 `投稿机会/征稿原文/{征文名}.md` |
| **与邕城朝暮兼容** | ✅ 可投 / ❌ 不可投（已投征文） |
```

## 邕城朝暮的投稿决策框架

**现状**：邕城朝暮已完成，准备投稿

**问题**：既想投征文（有奖金），又想投期刊（稿费+影响力）

**解决方案**：
1. **选择优先投稿目标**（征文 OR 期刊？）
2. **确认目标的首发政策**（邮件问）
3. **制定投稿序列**：
   - 如选征文 → 投一个征文 → 等结果 → 再投期刊
   - 如选期刊 → 投《红豆》→ 等发表 → 再投其他期刊 → 标注转载
