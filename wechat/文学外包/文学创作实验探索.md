## 实验方向001：对抗生成（Adversarial Literary Generation）

**状态**：构想阶段，未实验

### 动机

科普方法论的 Pass 3 Review 是合作博弈——Reviewer 帮文章变好，跟 Writer 目标一致。但文学写作有一个天然的对抗面：**"像AI写的"是死刑**。这构成了真正的对抗条件。

### AI 文学写作的六大败相（判官的武器）

对抗有意义的前提是判官能说清 AI 文字到底"假"在哪：

1. **匀质感**——每段质量差不多，没有真正的峰和谷。人类作家有神来之笔和过渡性废话，AI 全程中上，反而不真
2. **安全的洞察**——看起来深刻实则"正确的废话"。"城市的生命力在于人民"——对，但谁不知道？
3. **对称强迫症**——总忍不住搞对仗、排比、三段式。结构太工整 = 不像人
4. **情感宣告而非呈现**——"这让人震撼" vs 直接写那个震撼的画面让读者自己震撼
5. **万能连接词**——"然而""不仅如此""更重要的是"，AI 的过渡永远丝滑，人类经常硬切
6. **没有真正的冒犯性**——AI 不会写出让人不舒服的东西。文学需要偶尔的冒犯

### 三层架构设计

不是简单两个 Agent 对打，而是三层——防止 mode collapse（Writer 为逃避判官而牺牲文学品质）：

```
┌─────────────────────────────────────┐
│  Writer（写作者）                     │
│  目标：写出金线以上的文学作品          │
│  输入：素材 + 金线 + 上一轮判官反馈    │
└──────────────┬──────────────────────┘
               │ 稿件
               ▼
┌─────────────────────────────────────┐
│  Adversary（判官/鉴别者）             │
│  目标：判定这是不是 AI 写的           │
│  武器：六大败相清单 + 金线标杆文库      │
│  输出：逐句标注"AI味"浓度 + 判决理由   │
└──────────────┬──────────────────────┘
               │ 判决 + 逐句标注
               ▼
┌─────────────────────────────────────┐
│  Arbiter（仲裁者）                    │
│  目标：防止 Writer 为逃避判官而走偏    │
│  检查：金线标准是否还在？为了"不像AI"   │
│       有没有牺牲文学品质？             │
└──────────────┬──────────────────────┘
               │ 最终裁定：再改 / 通过
               ▼
            下一轮 or 终稿
```

**Arbiter 存在的原因**：文字版 mode collapse——Writer 学会故意写得粗糙、加错别字、逻辑不连贯来"骗过判官"。Arbiter 守住金线底线：可以不像 AI，但不能低于金线。

### 收益曲线预判

- **前几轮收益巨大**：判官指出匀质感、安全洞察、对称强迫症等 AI 通病，Writer 改掉后质量飞跃
- **中期收益递减**：明显 AI 味去掉后，剩下的差距是"机器没有真实体验"——对抗解决不了
- **后期可能有害**：Writer 为了"不像 AI"开始刻意反模式，文章变做作

### 初步结论

不做常态化对抗。把判官作为**金线自检的增强环节**——Phase 5 跑一轮对抗扫描，重点检六大败相，标注后让 Writer 针对性重写。一到两轮即可，不无限迭代。
