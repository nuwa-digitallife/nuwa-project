# 素材域：城市文化-南宁

从「邕城朝暮」实验提取的可复用核心素材。

## 核心意象

- **朱槿花**：朝开暮落，一株日开数百朵。嵇含《南方草木状》304年记录。南宁市花（1985年选定，87%投票）
- **邕江**：左江+右江在三江坡汇合。徐霞客记"自南宁来，过右江口岸，山始露石"

## 历史节点（有画面感的）

- 城墙：唐筑→宋加→清修→民国拆（1927-1946三十年拆完）→2004年修路翻出112米
- 龙象塔：明万历建→1937年怕成轰炸坐标炸毁→1985年重建，72铜铃
- 名字：晋兴→宣化→邕州→南宁，"邕"字贞观六年给，邕溪已不在
- 侬智高1052年：翻出自己未上报的归附奏章（人性荒诞）
- 狄青元宵夜袭昆仑关（电影感）
- 平话：狄青士兵留下的方言，至今城南有平南村、平西村
- 王阳明嘉靖七年建敷文书院讲心学，之后广西中举人数激增
- 邓小平1929年洋关码头上岸，一月后百色起义
- 1939年日军占南宁，昆仑关反攻

## 文化素材

- **语言混血**：壮语语序（树大、花红）→南宁白话（友仔、友女）
- **老友粉**：茶馆友仔病倒→老板炒蒜末豆豉辣椒酸笋碎肉下粉→"老友常来"
- **酸嘢**：徐霞客"以糟芥为案，山家清供"→穷但讲究
- **三月三歌圩**：光绪武鸣"沿江上下数里仕女如云"，嘹歌五言四句押腰脚韵
- **壮锦**："天纹之页"（壮语名）
- **铜鼓**：十二芒太阳纹+蛙（雷神之子）+谷穗纹带
- **五色糯米饭**：枫叶染黑、黄花汁染黄、黄栀子染紫
