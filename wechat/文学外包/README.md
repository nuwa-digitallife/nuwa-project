# 文学创作外包计划

用金线压缩-展开法批量生产文学作品，大规模搜索投稿机会并系统性投稿。把文学创作当项目跑。

---

## 一、核心思路

公众号是科普赛道的输出口。文学投稿是文学赛道的输出口。两条线并行：

```
科普线：选题 → 素材 → 多Pass写作 → 公众号发布 → 数据反馈
文学线：征稿搜索 → 选题匹配 → 金线写作 → 投稿 → 采用反馈
```

文学线的独特价值：
- **外部验证金线方法论**——被专业刊物采用 = 方法论 work 的硬证据
- **多体裁实验**——散文、小说、诗歌、报告文学，每种体裁验证金线法的适用边界
- **积累发表记录**——为后续出版/影响力打基础

## 三、选题-投稿匹配策略

不是有什么征稿就写什么。反过来——**先建素材库，再用素材匹配征稿**：

### 3.1 素材库建设（长期积累）

按主题域积累可复用的深度素材：

| 素材域 | 说明 | 可匹配的征稿方向 |
|--------|------|-----------------|
| **城市文化** | 南宁、广西、西南城市的历史文化人文 | 地方文学期刊、文化征文 |
| **科技人文** | AI/机器人/技术变革中的人性故事 | 科技散文、非虚构 |
| **非遗民俗** | 壮族文化、岭南民俗、手工艺 | 民族文学、非遗征文 |
| **红色历史** | 百色起义、昆仑关、南方革命史 | 红色题材征文 |
| **乡土中国** | 乡村变迁、城镇化、方言消亡 | 乡土文学、乡村振兴征文 |
| **跨文化** | 中西文化碰撞、海外华人、东盟 | 跨文化题材 |

### 3.2 匹配逻辑

```
新征稿机会进来
    ↓
匹配素材库 → 有高度匹配的素材域？
    ↓ 是                    ↓ 否
素材充足，直接进入          评估：值得专门采集素材吗？
金线压缩-展开流程               ↓ 是          ↓ 否
                          启动素材采集       跳过
```

优先投稿匹配度高的机会——素材库已有深度积累，写起来快且质量高。

## 四、批量投稿工作流

### 4.1 单篇流程

```
1. 征稿分析（5min）：体裁、字数、主题、截止日期
2. 素材匹配（10min）：现有素材库是否够？需要补采？
3. 金线压缩（关键步骤）：一个意象 + 一个洞察
4. 写作（按文学方法论 Phase 3-4）
5. 金线自检（Phase 5）
6. 格式化 + 投稿
7. 记录投稿状态
```

### 4.2 批量节奏目标

| 阶段 | 目标 | 说明 |
|------|------|------|
| **冷启动（Month 1-2）** | 搜索50+投稿机会，投稿5篇 | 验证流程，积累经验 |
| **稳定期（Month 3-6）** | 每月投稿3-5篇 | 多体裁、多刊物覆盖 |
| **优化期（Month 6+）** | 根据采用率调整策略 | 聚焦高命中率的刊物和题材 |

## 五、反馈闭环

### 5.1 数据追踪

| 指标 | 说明 |
|------|------|
| 投稿总数 | |
| 采用数 / 采用率 | 核心指标 |
| 平均响应周期 | 投稿到回复的天数 |
| 按体裁采用率 | 散文/小说/诗歌分别的命中率 |
| 按刊物级别采用率 | 国家级/省级/地方级 |
| 按素材域采用率 | 哪个主题域命中率最高 |

### 5.2 经验回传

每次收到采用/拒稿通知后：
- 采用 → 分析为什么成功，回传到文学方法论
- 拒稿 → 分析可能原因（主题不匹配？质量不够？风格不对？），回传到方法论或实验探索
- 积累10篇以上投稿数据后，做一次系统性复盘
