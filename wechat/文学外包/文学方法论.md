# 文学方法论

文学类写作的方法论和规则。与「内容方法论」（科普类）平行，同属公理层。

> **定位差异**：科普方法论的核心是"超越人类洞见"——机器智能的展示窗口。文学方法论的核心是"金线以上"——机器感受力的实验场。科普证明机器比人知道得多，文学实验机器能不能"看见"人性。

## 一、核心哲学：金线压缩-展开法

来源：冯唐金线说。核心判断——文学有一条客观的质量分界线，明眼人一清二楚。

**大模型对金线的独特价值**：
- 人类作家靠天赋和几十年阅读积累"金线感"，大模型天然读过金线以上的所有作品
- 大模型可以从海量素材中"压缩"出金线——一个核心意象 + 一个核心洞察
- 然后按照金线标准展开，每一句都在线上

**写作公式**：
```
海量素材 → 压缩成一句话（金线） → 按金线标准展开全文
```

> 🔺 **铁律：先压缩，后展开。** 不是边写边想，是先想透了再写。金线没压缩出来之前，一个字都不写。金线 = 一个核心意象 + 一个核心洞察，用一句话说清。

> 🔺 **铁律：素材涌现，不是预设。** 与科普方法论一致——不是带着框架找素材，而是素材足够丰富后，核心意象和洞察自然涌现。邕城朝暮的"朝开暮落"不是预设的，是在朱槿花、城墙、龙象塔、方言等素材反复出现"出现-消失-再出现"的模式后自然涌现的。

## 二、金线标准体系

### 内容金线（写什么）

| 标准 | 说明 | 自检问题 |
|------|------|----------|
| 冲击认知 | 不是重复常识，是揭示被忽略的真相 | 读者看完会说"我没想到"吗？ |
| 人性幽微 | 深入人性的暗角和亮角，不停留在表面情节 | 有没有一个让人沉默的细节？ |
| 真实 | 所写所思必须是真的，不矫情不虚饰 | 能删掉的煽情都删了吗？ |
| 往上引 | 不迎合读者，不往下坠 | 这段是在讨好还是在说真话？ |

### 形式金线（怎么写）—— 6C标准

| 标准 | 中文 | 检查 |
|------|------|------|
| Concise | 简约 | 每个字都必要吗？能删吗？ |
| Clear | 清澈 | 一读即懂吗？ |
| Complete | 完整 | 线索断了吗？ |
| Consistent | 一致 | 核心意象贯穿了吗？语调前后统一吗？ |
| Correct | 正确 | 历史事实、引文、年份都核实了吗？ |
| Colorful | 生动 | 有画面吗？有温度吗？ |

### 态度金线（用什么姿态写）

冯唐三要素：
- **感受在边缘**：不待在舒适区，触及边缘感受——"城墙不知道自己是遗迹"
- **理解在高处**：站在高处俯瞰——从1700年看一座城，从朱槿花看生命力
- **表达在当下**：用当下的语言写，不模仿古人，不追逐西方

## 三、写作流程

### Phase 0：征稿/选题分析
- 解读征稿要求：体裁、字数、主题范围、投稿格式
- 确定体裁选择（散文/小说/诗歌/报告文学）
- 锁定主题切口——不是"写南宁"，是"写南宁的什么"

### Phase 1：双线并行素材采集
两个方向同时搜索，互不干扰：

| 线 | 内容 | 目的 |
|---|---|---|
| **理论线** | 目标文学流派/标杆作家的写作理论和风格分析 | 建立金线标准 |
| **材料线** | 主题相关的历史、地理、人物、典故、语言、感官细节 | 提供涌现的原料 |

> 关键：材料线要搜有**画面感、情感温度、人性张力**的细节，不是百科词条式的知识罗列。一个"侬智高翻出未上报奏章"比"1052年侬智高攻邕州"值一百倍。

### Phase 2：金线压缩
从全部素材中提炼：
1. **核心意象**：一个贯穿全文的具象物（朱槿花、邕江、铜鼓……）
2. **核心洞察**：一个关于人性/世界的发现，用一句话说清
3. **金线语句**：意象 + 洞察合成一句话

示例：
```
意象：朱槿花的"朝开暮落"
洞察：城市的生命力不在城墙（会消失的），在每天都在发生的事（花开、粉煮、歌唱）
金线：消失的和留下的，从来不是你以为的那些
```

压缩完成的标志：这句话能让你自己想写。如果不能，说明还没找到。

### Phase 3：结构设计
以核心意象为脊柱，搭建结构：
- **开篇**：直入意象。不要铺垫，不要"自古以来"
- **展开**：每个段落是意象的一次变奏——不同时空、不同载体呈现同一个"朝开暮落"
- **呼吸点**：独立短句做节拍器（"花比城早。""历史在这个细节上格外沉默。"）
- **收束**：回到意象，但比开篇更深一层。"它不管朝代更迭。它只管开。"

结构法则：
- 一个核心意象贯穿，不换
- **对位结构**：消失的 vs 留下的、安静 vs 不安静、拆 vs 建。成对出现，产生张力
- 时间线不必顺序，可以跳跃，但每次跳跃都要靠意象接住

### Phase 4：展开写作

#### 语言铁律

> 🔺 **铁律：陈言务去。** 韩愈原话。去除一切陈旧老套的表达。宁可用最朴素的大白话，也不用"优美"的烂俗句。

> 🔺 **铁律：动词驱动，形容词极少。** 冯唐风格的核心——用动词推叙事，用名词造画面。形容词能删就删，不能删就转化为动作。"城墙筑了拆"比"城墙历经沧桑"好一百倍。

> 🔺 **铁律：短句是锤子，长句是河流。** 关键洞察用短句砸出来。叙事用长句流过去。节奏感 = 短长交替，不是全短也不是全长。

> 🔺 **铁律：引文原味保留。** 古诗词、历史文献、方言俗语——原文引用，不翻译不解释。嵇含写的就是嵇含写的，不要"意思是说"。读者看不懂的地方，用上下文暗示，不用括号注释。

#### 陈言务去清单（禁用表达）

以下表达出现在文学散文中 = 直接不及格：

| 类别 | 禁用词/句 | 替代思路 |
|------|-----------|----------|
| 空洞赞美 | 源远流长、博大精深、灿烂辉煌、璀璨夺目 | 用具体事实说话 |
| 假大空 | 自古以来、众所周知、毋庸置疑 | 直接进入 |
| 套路抒情 | 不禁让人感慨、令人心潮澎湃、让我们一起 | 删掉，让事实自己说话 |
| 导游词 | 美丽的XX、壮观的XX、雄伟的XX | 换成具体动作或画面 |
| 公文体 | 在...的推动下、为了...的发展、深入贯彻 | 这不是散文该出现的 |
| 鸡汤 | 岁月静好、诗和远方、不负韶华 | 宁可沉默也不用 |
| 感叹号 | 多美啊！太震撼了！ | 去掉感叹号，能用句号的绝不用感叹号 |

### Phase 5：金线自检

完成初稿后，逐条检查：

**A. 金线压缩回测**
- [ ] 全文能否用一句话概括？（不能 = 散了）
- [ ] 核心意象是否贯穿始终？有没有中间断掉？
- [ ] 核心洞察是否在文中自然浮现？（不是喊出来的）

**B. 6C逐条**
- [ ] Concise：全文逐段，每段是否可以删减？删减后是否影响完整性？
- [ ] Clear：有没有读者需要读两遍才懂的句子？
- [ ] Complete：有没有线索断了没接上的？
- [ ] Consistent：语调/视角有没有跳？核心意象有没有被新意象抢戏？
- [ ] Correct：所有年份、人名、引文、历史事件核实
- [ ] Colorful：有没有连续三段以上"干燥无画面"？

**C. 陈言务去扫描**
- [ ] 全文搜索禁用词清单，发现即替换或删除
- [ ] 检查形容词密度——连续出现两个以上形容词的句子，考虑重写

**D. 短句呼吸检查**
- [ ] 全文是否有节奏感？是否有足够的"呼吸点"？
- [ ] 独立短句是否放在了真正重要的位置？（不要滥用）

**E. 合规检查**
- [ ] 字数是否符合征稿要求
- [ ] 体裁是否对标
- [ ] 投稿格式是否正确

## 四、与科普方法论的关键差异

| 维度 | 科普方法论 | 文学方法论 |
|------|-----------|-----------|
| **核心命题** | "你写的像AI写的"是最高评价 | "这不像AI写的"才是最高评价 |
| **信息密度** | (事实数+观点数)/字数 越高越好 | 密度不是目标，节奏和留白才是 |
| **框架** | 读者可带走的分析工具 | 读者带走的是一个意象、一种感受 |
| **涌现对象** | 框架涌现 | 意象涌现 |
| **人设** | 三体角色体系，多人设轮换 | 待建——可能需要文学人设体系 |
| **语言标准** | 清晰准确，专业概念必须祛魅 | 精准但允许留白，引文不翻译 |
| **Review重点** | 事实核查、春秋笔法、框架去重 | 金线自检、陈言务去、节奏感 |
| **素材采集** | 多源交叉验证，标注信源 | 找有画面感和人性张力的细节 |
| **成功标准** | 获得感+话题感+信息浓度 | 金线以上+有沉默的瞬间+读完放不下 |

## 五、实验记录

### 实验001：邕城朝暮

**基本信息**

| 项 | 值 |
|---|---|
| **目标刊物** | 《红豆》杂志·南宁文化历史专题 |
| **体裁** | 散文 |
| **字数** | ~4300字 |
| **终稿位置** | `~/Desktop/邕城朝暮.docx` |
| **待验证** | 是否被刊物采用；读者反馈；金线标准是否可复用于其他文学体裁 |

#### 完整写作过程复盘

**Step 1：征稿分析**

征稿来源：《红豆》杂志南宁文化历史专题。体裁要求散文不超过5000字。主题范围：古代历史、近现代事件、红色故事、城市变迁、人文景观、非遗民俗。投稿邮箱 hdzz1972@163.com，格式"体裁+标题+姓名"。

决策：选散文，控制在4000-4500字。主题不预设，等素材涌现。

**Step 2：双线并行素材采集**

两个 Agent 同时启动，互不干扰：

- **理论线 Agent**：研究冯唐金线说。产出——金线的核心标准（内容要冲击认知+探索人性幽微，形式要陈言务去+挑战语言效率极限）、6C标准（Concise/Clear/Complete/Consistent/Correct/Colorful）、冯唐散文风格特征（动词驱动、形容词极少、短句密集、古典与现代熔铸）、三要素（感受在边缘/理解在高处/表达在当下）。
- **材料线 Agent**：搜集南宁文化历史素材，按10个维度展开——古代建城史、地理文化、非遗民俗、美食、城市变迁、人文景观、植物自然、诗词文献、红色历史、方言语言。产出约100条具体素材，特别标注了有画面感和人性张力的细节。

关键：材料线不是在搜"百科知识"，是在搜**"能让人停下来的细节"**。比如：
- 侬智高在知州军资库里翻出自己未被上报的归附奏章（历史的荒诞）
- 狄青元宵夜酒杯还热着就悄悄传令夜袭昆仑关（电影感）
- 金狮巷"叮叮当当"打金银器的声音（听觉记忆）
- 南宁小孩吮朱槿花蜜（身体记忆）
- 壮语"红花"说"花红"，先看见事物再说形容（语言哲学）
- 嵇含304年记录朱槿"日开数百朵，朝开暮落"（花比城早）

**Step 3：金线压缩**

素材全部摊开后，反复出现的模式浮现了——**"出现-消失-再出现"**：
- 城墙筑了拆，拆了埋，埋了又翻出来
- 龙象塔建了炸，炸了又修回来
- 名字换了四次，但"邕"字一直在
- 军队消失了，方言留下来了
- 铺子关了，安静本身成了记忆
- 朱槿花朝开暮落，落了第二天接着开

这个模式跟朱槿花的"朝开暮落"完美同构。核心意象锁定。

然后压缩成一句话（金线）：

> **南宁像朱槿花，朝开暮落——城墙拆了、塔炸了、名字换了四次，但花还在开，粉还在煮，歌还在唱。一座城的生命力不在城墙上，在那些每天都在发生的事上。**

拆解：
- **核心意象**：朱槿花的"朝开暮落"
- **核心洞察**：消失的和留下的，从来不是你以为的那些
- **情感内核**：不悲壮，不赞美，只是看见——它不管朝代更迭，它只管开

压缩完成的感觉：这句话让我想写。想把它展开。

**Step 4：结构设计**

以"朝开暮落"为脊柱，每个段落是这个意象的一次变奏：

```
嵇含记朱槿（304年）→ 吮花蜜（感官锚点）
    ↓
城墙筑了拆（物理的朝开暮落）
龙象塔建了炸（又一轮）
名字换了四次（连名字也是）
    ↓
"但南疆从来不安宁"（转折——深入历史）
侬智高翻出奏章（人性的荒诞——全文最沉的一笔）
狄青元宵夜袭（历史的戏剧性）
军队消失 vs 方言留下（对位：消失的和留下的）
    ↓
语言混血（留下的东西之一）
老友粉（留下的东西之二——人间温度）
酸嘢 + 徐霞客（"穷，但讲究"）
    ↓
王阳明建书院（"一个人来了，一座城开了"）
扬美古镇（打仗的人走了，读书的种子留下了）
金狮巷（声音消失，安静成了记忆）
    ↓
三月三歌圩（活着的非遗）
壮锦"天纹之页"（四个字本身是诗）
铜鼓 + 五色糯米饭（铸造的希望）
    ↓
邓小平洋关码头（红色历史——又一个从这里开始的故事）
昆仑关大捷（战火——又一轮落）
"我猜是煮一碗老友粉"（战后——接回人间）
    ↓
"城市的生命力从来不在城墙上"（洞察浮现）
绿城 + 扁桃 + 朱槿（回到意象）
    ↓
"墙是用来隔开的。花是用来打开的。"（金句收束）
"它不管朝代更迭。它只管开。"（终句——回到304年，闭环）
```

设计原则：
- 对位结构贯穿——拆vs建、消失vs留下、安静vs不安静、墙vs花
- 独立短句做呼吸点，放在最重要的位置（"花比城早。""历史在这个细节上格外沉默。""它只管开。"）
- 时间线不顺序——304年→唐宋→1052年→明→1929年→1939年→当代→回到304年。靠意象而非时间串联

**Step 5：展开写作**

遵循的语言铁律：
- **动词驱动**："城墙筑了拆""凤凰没压住，塔先飞了""攥在一起不散"
- **形容词极少**：全文几乎没有"美丽的""壮观的""雄伟的"
- **引文原味**：嵇含、徐霞客、光绪武缘县图经、历代邕州诗——直接引，不翻译
- **陈言务去**：没用"源远流长""博大精深""灿烂辉煌"，没用感叹号
- **短句是锤子**："花比城早。""他本来要做臣子的。""这四个字本身已经是诗了。"

**Step 6：金线自检**

| 6C标准 | 检查结果 |
|---|---|
| Concise | 全文约4300字，无废段。每段一个意象或事件 |
| Clear | 短句为主，动词叙事，无晦涩 |
| Complete | 从318年建城到当代博览会，1700年完整 |
| Consistent | "朝开暮落"贯穿始终，语调前后一致 |
| Correct | 所有历史年份、人名、引文均有出处核实 |
| Colorful | 侬智高奏章、吮花蜜、金狮巷叮当声——有画面有温度 |

#### 关键经验

1. **素材量决定涌现质量**：两个Agent并行搜了约100条素材，"朝开暮落"的模式才浮现。如果只搜了20条，大概率涌现不出这个意象。
2. **最好的细节不是最壮观的，是最反常的**：侬智高那封没上报的奏章，比任何宏大叙事都有力。因为它揭示人性——"不是因为他想反，是因为没人听他说话"。
3. **压缩是最难的一步**：素材有了，结构好设计，语言可以打磨。但"压缩成一句话"这一步，要么涌现要么没有，不能硬凑。
4. **独立短句要克制**：一开始写了太多独立短句，后来删了一些。短句是锤子，用多了就不震了。
5. **结尾回到开头但更深一层**：开头是嵇含304年记花，结尾也是304年——但加了"那年司马氏的天下已经乱成一锅粥"的背景。同一个时间点，读者第二次读到时理解完全不同。

## 六、开放问题（待后续实验回答）

1. **文学人设体系**：科普有六个三体角色，文学需要什么样的声音体系？冯唐式短句？沈从文式白描？汪曾祺式闲笔？还是涌现自己的声音？
