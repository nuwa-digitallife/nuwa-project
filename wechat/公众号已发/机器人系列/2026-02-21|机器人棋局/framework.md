# 排名框架设计：五维棋力模型

## 核心洞见

所有24篇素材文章都在回答"谁最火"，没人回答"谁最强"。
"最火"看的是春晚曝光、融资额、媒体声量；
"最强"看的是能不能活下来并赢下来。

我们的框架不排"谁最火"，排"谁最可能赢"。

## 五维棋力

### 1. 造血（Revenue Power）
- 有没有真金白银的收入？
- 盈利还是烧钱？
- 收入来源是机器人本身还是其他业务交叉补贴？
- 权重最高——因为这是唯一不能靠讲故事获得的维度

### 2. 技术壁垒（Tech Moat）
- 自研率（硬件+软件）
- 是否有独创技术路线（vs 跟随主流）
- 论文/专利数量和质量
- 模型是否开源（开源=生态影响力，但也=壁垒降低）

### 3. 规模化能力（Scale Readiness）
- 实际出货量（不是"产能"或"订单意向"）
- 供应链掌控力（自研核心零部件 vs 外购）
- 成本控制能力
- 量产交付的可重复性

### 4. 生态位清晰度（Strategic Clarity）
- 是否清楚自己在打什么仗？
- C端/B端/平台/垂直——定位是否明确且可执行
- 产品线是聚焦还是发散
- 这一维度筛掉"什么都想做但什么都没做好"的公司

### 5. 资本弹药（Capital Runway）
- 融资总额和现金储备
- 估值是否合理（高估值≠强，可能是下一轮融资的障碍）
- 投资者质量（产业资本 vs 纯财务VC）
- 上市进程（IPO=最大弹药补给）

## 评分逻辑
- 每个维度 0-10 分
- 造血权重 x2（因为这是终极验证）
- 总分 = 造血x2 + 技术 + 规模 + 生态位 + 资本
- 满分 60

## 排名结果预设（基于素材数据）

### T0: 盈利且规模化
- 宇树科技：造血9(连续5年盈利/10亿营收) + 技术8(95%自研) + 规模9(23700+1500台) + 生态位7(四足称王但人形刚起步) + 资本8(IPO在即)

### T1: 高估值高订单但未盈利
- 银河通用：造血3(有订单无公开营收) + 技术8(VLA最强学术团队) + 规模5(百台级) + 生态位7(零售场景明确) + 资本9(行业最高估值)
- 智元机器人：造血4(13亿订单但未公开营收) + 技术7(ViLA+稚晖君) + 规模8(5000+台) + 生态位7(工业实用主义) + 资本8(港股在即)
- 优必选：造血5(13亿营收但亏11.6亿) + 技术6(全栈+华为合作) + 规模7(千台下线) + 生态位6(工业+消费并行) + 资本7(已上市但亏损)

### T2: 特色鲜明的挑战者
- 魔法原子、松延动力、傅利叶、星海图、逐际动力

### T3: 潜力股/早期
- 它石智航、原力灵机、开普勒、禾芯动力

### 生态玩家(不直接排名)
- 华为(平台)、小米(内部验证)、追觅(孵化器)

## "超越人类的洞见"在哪？

1. **框架本身**：没有人做过这样的多维评价，所有文章都是堆事实或单维度比较
2. **造血权重x2**：揭示行业真相——大部分公司在用融资补贴亏损，只有宇树真正赚钱
3. **达闼的警示**：曾估值223亿的先驱现在欠薪裁员——高估值≠高存活率
4. **春晚ROI的本质**：不是卖货而是卖故事给投资人
5. **预测**：2026-2027将有一波洗牌，类比2019新能源——活下来的不超过5家
