# 深度盘点机器人竞争格局（一）：谁才是国内最强？

春晚结束后三天，我做了一件事：把24篇媒体报道的关键词做了统计。出现最多的词是"未来"。出现最少的词是"利润"。

**所有人都在说"谁最火"，没有一个人回答"谁最强"。**

"最火"看的是曝光量、融资额、春晚露出时长。"最强"看的是——**能不能活下来，并且赢下来。**

这篇文章不排"谁最火"。我们排"谁最可能赢"。

要排名，先得定义什么叫"强"。

## 五维棋力：一个给机器人公司打分的框架

评价一家机器人公司，不能只看单一维度。技术强不代表能量产，能量产不代表能赚钱，能赚钱不代表赛道选对了。

我们提出一个"五维棋力"模型，五个维度，每个0-10分，其中造血能力权重翻倍——因为这是唯一不能靠讲故事获得的东西。

**造血（能不能赚到钱）**。有真金白银的收入吗？是盈利还是在烧钱？收入来源是机器人本身，还是其他业务在交叉补贴？这个维度的权重是其他维度的两倍。原因很简单：融资可以讲故事拿到，技术可以招人追上，产能可以砸钱建起来——但营收，只有客户愿意掏钱才算数。

需要说明：**这个框架天然偏向已有收入的公司。** 如果你认为技术路线选择比短期营收更重要，这个排名会完全不同。但我们的立场是：在一个即将进入洗牌期的赛道上，活着是赢的前提。

**技术壁垒（有没有别人抄不走的东西）**。硬件自研率、独创技术路线、论文和专利的含金量。一个微妙的点：模型开源意味着生态影响力，但也意味着壁垒在消融。

**规模化能力（能不能批量交付）**。看的是实际出货量——不是"产能"，不是"订单意向"，不是"计划"。供应链掌控力、成本控制、量产的可重复性，都在这一维度。

**生态位清晰度（知不知道自己在打什么仗）**。C端消费、B端工业、平台生态、垂直场景——定位是否明确且可执行。这一维度专门筛掉"什么都想做但什么都没做好"的公司。

**资本弹药（还能撑多久）**。融资总额、现金储备、投资者质量。一个反直觉的点：高估值不等于强。过高的估值反而可能成为下一轮融资的障碍——没有人愿意接盘一个涨不动的估值。

> **总分 = 造血×2 + 技术 + 规模化 + 生态位 + 资本。满分60。**

框架定好了，逐个过。

## T0：唯一赚到钱的选手

**宇树科技**是目前唯一一家在机器人业务上实现盈利的公司。

2024年营收超过10亿元，自2020年起连续五年盈利——在一个烧钱如流水的赛道上，这个成绩近乎异类。

但它赚钱的主力不是人形机器人，而是四足机器狗。2024年四足出货23700台，拿下全球69.75%的市场份额。人形机器人方面，2024年交付超过1500台，已经是国内交付量前三。**我们把四足和人形视为同一家公司的整体竞争力——因为四足的利润是它做人形的弹药。**

技术层面，宇树走的是"硬件为王"路线：自研率超过95%，电机、减速器、激光雷达全部自己做，专利申请超过454项，已授权268项。这意味着它对供应链的掌控力远超同行——别人还在为核心零部件的产能发愁，它已经在优化良品率了。

2026年春晚已经是它第三次登台。24台G1和H2混合编队表演《武BOT》，H2首次亮相。播出两小时内京东搜索量暴涨300%，订单增长150%，春晚同款（G1 EDU进阶版，20.9万元）几分钟售罄。

创始人王兴兴，1990年生，上海大学机械硕士，在大疆短暂任职后创业。IPO辅导已于2025年11月完成，正在推进科创板上市。

## T1：估值百亿但还没赚到钱

第二梯队是三家估值百亿级、技术和订单都很硬，但还没有公开证明赚钱能力的公司。

**银河通用**是资本市场最看好的那个。

成立不到三年，估值已经冲到30亿美元（超200亿元），是国内人形机器人赛道最高估值。三年估值翻了70倍。最新一轮融资超3亿美元，创下单轮融资纪录。

创始人王鹤是北大助理教授、斯坦福博士，团队在CVPR、NeurIPS等顶会发表过50余篇论文。技术路线是VLA——直接用大模型驱动机器人的视觉、语言和动作，目前推出了GraspVLA、GroceryVLA等针对不同场景的专用模型。

商业化方面，已签约100家智能零售门店，与百达精工签了1000台以上的战略订单。春晚中它是"指定具身大模型机器人"，在贺岁微电影《我最难忘的今宵》中与沈腾马丽同台。

但一个关键数字缺失：**营收没有公开披露**。在200亿估值之下，这个沉默很耐人寻味。

[插图3：radar_t1.png — T1梯队三家对比雷达图]

> **棋力评分：造血3×2 + 技术8 + 规模5 + 生态位7 + 资本9 = 35/60**

**智元机器人**走了一条截然不同的路。

创始人彭志辉（"稚晖君"）从B站百大UP主、华为天才少年转身创业，实控人邓泰华出身华为高管层。这个组合本身就是一面旗帜——技术极客加产业老兵。

智元的标签是"实用主义"。创始人公开表态"不玩后空翻"，所有精力砸在连续操作能力上。2024年12月启动量产，2025年1月第1000台下线，到年底突破5100台——这是公开数据中国内最高的人形机器人出货量。

但另一个数据也值得注意：智元没参加春晚。CMO说"预算有限，更愿意把钱投研发"。它选择在除夕当晚自办了一场"机器人奇妙夜"，200多台机器人演了12个节目。这个选择本身就说明了一种性格——**不争曝光，争交付。**

2025年订单突破13亿元（龙旗科技数亿、均胜电子过亿、中国移动7800万等）。已完成B+轮融资，腾讯、京东、LG、正大集团均入场。市场传闻2026年第三季度冲击港股，但公司回应"暂无明确计划"。

> **棋力评分：造血4×2 + 技术7 + 规模8 + 生态位7 + 资本8 = 38/60**

**优必选**是这个赛道里资历最老、也最有争议的选手。

2012年成立，2023年底港股上市，是"人形机器人第一股"。Walker系列已迭代到S2，2025年12月第1000台在柳州下线。上市公司的好处是数据透明：2024年营收13.05亿元，同比增长23.7%。

坏处也是数据透明：**全年亏损11.6亿元，五年累计亏损超过50亿。** 更残酷的是，13亿营收中，人形机器人只贡献了3500万——其余来自消费级机器人和教育机器人。

2025年转机出现：全年订单突破14亿，与华为签署战略合作，16.65亿入主A股锋龙股份（收购后18连板）。优必选正在试图用上市公司的资本杠杆撬动更大的产业布局。

但问题很明确：**什么时候人形机器人本身能赚钱？**

> **棋力评分：造血6×2 + 技术6 + 规模7 + 生态位5 + 资本7 = 37/60**
>
> （注：优必选造血分高于智元，因为13亿是经审计的真实营收，而智元13亿是订单额——订单不等于到账。生态位扣分因为C端+B端+教育多线并行，聚焦度不足。）

## T2：特色鲜明的挑战者

第三梯队的公司各自有一张独特的牌。

**松延动力**打的是"价格破局"。一台人形机器人9998元——这个定价在行业里炸开了锅。产品叫"小布米"，京东上线一小时卖出100台，首批500台两天售罄。创始人姜哲源是清华博士辍学的95后，放话"人形机器人以后能论斤卖"。春晚小品《奶奶的最爱》中，32个电机让机器人像素级复现了蔡明的脸。估值超20亿，在四家春晚参演企业中是唯一面向C端消费者的。

> 造血3×2 + 技术5 + 规模5 + 生态位8 + 资本6 = **27/60**

**星海图**的牌是"学术天团"。两位清华交叉信息研究院助理教授联合创始（赵行、许华哲），在国内具身智能公司中学术底蕴最厚。产品R1定价19.9万元，走轮式路线。融资近20亿，估值近百亿。短板很明显——商业化太早期，40多家客户相比同行的千台级订单还差一个量级。

> 造血2×2 + 技术9 + 规模4 + 生态位6 + 资本9 = **32/60**

**傅利叶智能**的牌是"康复基因"。2015年从医疗康复机器人起家，对人体运动学的理解是大多数做AI的团队不具备的。GR系列已迭代到第三代Care-bot，是国内首款可商业化量产交付的双足人形机器人。估值约80亿，E轮融资近8亿，已完成股改备上市。

> 造血3×2 + 技术6 + 规模6 + 生态位8 + 资本7 = **33/60**

**逐际动力**最值得关注的是它的生态绑定。京东战略领投，意味着零售、物流、服务场景有了明确的落地通道。累计融资超20亿，2026年2月刚完成2亿美元B轮。创始人张巍是南科大长聘教授，学术根基扎实。

> 造血2×2 + 技术7 + 规模4 + 生态位7 + 资本8 = **30/60**

**魔法原子**背靠追觅科技，硬件自研率超90%，春晚宜宾分会场表演了托马斯回旋和侧空翻。技术炫酷，但独立估值未公开、独立营收未公开——追觅体系内的资源支持是优势，也是需要证明的独立性问题。

> 造血2×2 + 技术7 + 规模5 + 生态位6 + 资本7 = **29/60**

**开普勒机器人**走的是务实工业路线。创始人杨华出身小米生态链纯米科技，半年内引入5家上市公司股东，产业绑定程度最深。先行者K2单手负载15kg，已有数千台意向订单。

> 造血3×2 + 技术5 + 规模5 + 生态位7 + 资本6 = **29/60**

## 一个警示：达闼的故事

在排名之外，有一个不在任何排行榜上但所有人都应该知道的名字——**达闼机器人**。

创始人黄晓庆，前中国移动研究院院长，被誉为"云端机器人之父"。2015年成立，累计融资超54亿元，2023年C轮估值高达223亿元。

然后呢？

2025年3月，达闼爆出欠薪裁员危机。员工从巅峰800多人缩减到约150人，广州分公司停水停电关闭，上海、北京、深圳数百名员工陷入长达一年的讨薪纠纷。两次冲击上市失败。

**223亿估值，150人挣扎求生。** 这是整个行业最应该被记住的一课：高估值不等于高存活率，技术愿景不等于商业能力。

> 在机器人赛道上，死掉的公司不是因为技术不行，而是因为钱花完了还没找到付费场景。

## 看不见的玩家：大厂暗棋

不做本体的玩家同样在塑造格局。

**华为**明确说了"不做机器人本体"，但发布了CloudRobo具身智能平台、与优必选签了战略合作、哈勃投了千寻智能、东莞砸了72亿建产业园。它想做的是"机器人界的安卓"——让所有本体厂商跑在它的底座上。

**小米**已经把CyberOne迭代到第四代，但全部20多台都投入了亦庄汽车工厂做数据采集。雷军宣布未来五年研发预算2000亿（覆盖手机、汽车、AI全线业务）。小米的策略是：**不抢发布会，先在自己的工厂里把数据跑起来。** 谁拥有最大规模的真实工厂数据，谁就在下一轮竞争中占据先机。

## 技术路线：一场没有定论的分歧

在公司层面的竞争之外，还有一场更底层的技术路线之争。

**VLA（视觉-语言-动作）** 是当前主流。银河通用、魔法原子、智元、星海图都选了这条路。核心思路是用大语言模型驱动机器人——让机器人像人一样"看到、理解、行动"。但真机评测数据很冷酷：目前最强的VLA模型在标准化评测中成功率只有51%，环境稍有变化性能平均下降41%。

**世界模型** 是2026年最大的变量。英伟达在2月刚发布了DreamZero，用500小时数据实现了泛化能力2倍提升。Jim Fan称之为机器人领域的"GPT-2时刻"。国内的它石智航和原力灵机在探索这条路线，但需要GB200级别的算力——短期内只有大厂玩得起。

[插图5：tech_routes.png — 三大技术路线对比图]

**数据采集方式**也在分化。传统遥操作一套设备35万元，操作员连续工作只能撑两小时。它石智航用可穿戴手套把成本降低了一个数量级以上。智元的AgiBot World平台积累了超过100万条机器人轨迹数据。英伟达的DreamDojo用了44711小时的人类视频做预训练。

> **硬件正在标准化和低成本化，真正的护城河正在从"造得出来"转移到"谁的AI更聪明、谁的数据更多"。**

## 谁能活过洗牌期？

一个不可忽视的背景：国家在2023年发布了《人形机器人创新发展指导意见》，目标是2025年初步建立产业链、2027年达到世界先进水平。北京、深圳、芜湖等地方政府纷纷推出补贴和产业园政策。政策红利正是这么多公司同时存在的重要原因——但政策不会永远兜底。

坦率说，2026年的中国机器人赛道和2019年的新能源汽车极为相似。

摩根士丹利在研报中直接做了这个类比：彼时新能源车企上百家，补贴退坡后活下来的不超过十家。人形机器人目前融资超过1亿的公司就有二十多家，但2025年全行业16家公司的订单总额加起来只有57亿元。

**数十家公司分57亿元的蛋糕，平均每家不到4亿。** 扣掉硬件成本和研发开支，大部分公司的现金流是负的。

做一个不精确但有用的预测：**到2027年底，能作为独立实体继续运营的国内人形机器人公司，大概率不超过8家。** 其余的要么被收购，要么转型，要么像达闼一样挣扎在存亡线上。

谁能活下来？回到五维棋力模型——

**造血能力决定生存**。宇树凭借四足机器狗的利润有持续输血能力，智元和优必选靠大额订单可以撑到上市窗口，银河通用靠史上最大单轮融资可以再烧两年。其余公司需要在18个月内找到可持续的收入来源。

**生态位决定方向**。松延动力锁死C端万元级、傅利叶守住医疗康复、逐际动力绑定京东物流——这些生态位清晰的公司，即使规模不大，存活概率也高于那些"什么都做"的全面型选手。

**春晚的本质是一场豪赌**。四家参演企业每家投入约1亿元。这笔钱的ROI不在销量——春晚同款20万元的机器人卖不了几台——而在品牌曝光带来的下一轮融资便利。用1亿换一个更高的估值，再用更高的估值换更多的钱，用更多的钱撑到技术成熟。这是一条用讲故事争取时间的生存策略。

**但故事总有讲完的一天。**
