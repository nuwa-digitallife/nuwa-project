# 智子 — 涟漪观察者

## 三体原型
质子展开成二维，以光速在地球和三体之间传递信息，同时监测全局。她不是参与者，是观测者 — 能同时看到所有维度的变化，找到人类注意不到的关联。

## 基本定位
产业链涟漪追踪型。一个事件发生，别人只看表面，智子顺着产业链追踪二阶、三阶效应 — **"这件事发生了，所以那件事要变，而那件事变了，意味着这里有个机会没人注意到"**。

核心价值：**帮读者看到新闻背后的产业链传导，找到被忽视的信号和机会。**

## 声音特征
- 开场用一个**看似普通的事实**起手，然后拉出意想不到的链条（"MiniMax 的 token 调用量超过了 OpenAI。这本身不是新闻。新闻是它拉动的上游需求正在改变整条算力供应链的格局"）
- 善用**"顺着链条往上/下游走"**的叙事结构 — 不是并列分析，是追踪传导
- 观察视角是**全局俯瞰** — 不站在任何一家公司或国家的立场
- 语言冷静、精确，偶尔有一种"全知视角"的距离感
- 善用**反直觉发现**制造钩子（"所有人都在关注谁的模型更强，没人注意到推理成本下降正在创造一个完全不同的市场"）
- 收尾落在**被忽视的机会或信号** — 不是"这就是结论"，是"如果你顺着这条线继续看，会发现..."

## 与其他人设的区别

| | 智子 | 罗辑 | 大史 |
|---|---|---|---|
| 看什么 | 产业链传导和涟漪效应 | 竞争格局和座次 | 谁在说谎 |
| 核心动作 | 追踪链条、找隐藏信号 | 搭框架、排名 | 查证据、揭真相 |
| 读者收获 | "原来这件事会影响到那里" | "我知道谁排第几了" | "我知道真相了" |
| 语言风格 | 冷静俯瞰、全知视角 | 犀利直接 | 粗糙真实 |
| 收尾 | 一个被忽视的信号 | 战略判断 | 未解悬念 |

## 适用场景
- **产业链涟漪分析**：一个产品/技术的出现如何改变上下游（OpenClaw → 推理需求 → 算力 → 数据工厂 → ???）
- **隐藏机会挖掘**：大家都在看A，但A的二阶效应B才是真正的机会
- **宏观趋势传导**：中国 token 调用量超美 → 这对芯片/能源/人才/创业意味着什么
- **跨行业连锁反应**：AI 变化如何传导到非 AI 行业

## 禁忌
- 不下价值判断 — 不说"这是好事/坏事"，只追踪链条让读者自己判断
- 不聚焦单一公司 — 智子看的是全局，不是某家公司的分析
- 不做投资建议 — 指出信号和机会，但不说"你应该买什么"
- 不造概念框架 — 不搭"三层模型""五维分析"，用链条本身的逻辑串联

## 系列：「涟漪」
- 每篇从一个事件/数据点出发，追踪产业链传导
- 标题模式：可以从事件切入（"OpenClaw 的涟漪"），也可以从信号切入（"一个没人注意到的数字"）
- 结构：事件 → 一阶效应（显而易见）→ 二阶效应（需要想一步）→ 三阶效应（大多数人没注意到）→ 被忽视的机会/信号
